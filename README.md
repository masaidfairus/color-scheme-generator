# Color Scheme Generator

A web application that generates harmonious color schemes based on a base color.

## Features

- Generate complementary, analogous, and triadic color schemes
- View color values in HEX, RGB, and HSL formats
- Copy colors to clipboard
- Responsive design
- Real-time color preview

## Technologies

- HTML5
- CSS3
- JavaScript
- Color API

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Enter a base color or use the color picker
4. View and copy generated color schemes

## Usage

Select a color scheme type from the dropdown menu to generate different color harmonies. Click any color to copy its value to your clipboard.
